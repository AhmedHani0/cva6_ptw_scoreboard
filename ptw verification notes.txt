PTW state Machine :
typedef enum logic [2:0] {
  SB_IDLE,
  SB_WALKING,
  SB_WAIT_PTE,
  SB_EXPECT_UPDATE,
  SB_EXPECT_ERROR,
  SB_KILLED
} sb_state_e;

SB_IDLE
  No active tracked PTW transaction.

SB_WALKING
  A shared-TLB miss was accepted and a page walk is active.

SB_WAIT_PTE
  A page-table memory request was issued; scoreboard waits for returned PTE.

SB_EXPECT_UPDATE
  Scoreboard saw a legal leaf PTE and expects shared_tlb_update_o.valid.

SB_EXPECT_ERROR
  Scoreboard saw an invalid/illegal PTE and expects ptw_error_o or ptw_access_exception_o.

SB_KILLED
  Flush aborted the walk; scoreboard expects no useful update from this old walk.
